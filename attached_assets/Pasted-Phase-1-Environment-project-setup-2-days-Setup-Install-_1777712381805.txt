Phase 1 — Environment & project setup ~2 days
Setup

 Install PHP 8.2 on server / Hostinger VPS
 Install Composer globally
 Create new Laravel 11 project: composer create-project laravel/laravel laravel-api

Config

 Configure .env: DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD (same Hostinger MySQL)
 Test DB connection: php artisan migrate:status
 Configure CORS in config/cors.php — allow Vite origin (localhost:5000) and production domain
 Update Vite proxy in vite.config.ts: /api → http://localhost:8000
 Set SESSION_DRIVER=database in .env

Auth

 Install Laravel Sanctum: composer require laravel/sanctum
 Publish Sanctum config: php artisan vendor:publish --provider=Sanctum

Test

 Verify Laravel health check route GET /health returns 200
 Run php artisan session:table && php artisan migrate

Phase 3 — Authentication & middleware ~2 days
Auth

 Create AuthController::register() — hash password with bcrypt(10), create user, return session
 Create AuthController::login() — validate credentials, Auth::attempt(), return user JSON
 Create AuthController::logout() — Auth::logout(), invalidate session
 Create AuthController::user() — return authenticated user (GET /api/auth/user)
 Create AuthController::updateProfile() — PUT /api/auth/user
 Create AuthController::uploadProfileImage() — base64 decode + Storage::put
 Create AdminAuthController::adminLogin() — POST /api/admin/login

Middleware

 Create AdminMiddleware: check auth + user->isAdmin, return 403 JSON on fail
 Register AdminMiddleware in bootstrap/app.php as 'admin'

Routes

 Define auth routes in routes/api.php (no middleware group)

Test

 Test all auth endpoints match Express response shape exactly

Tests

Cutover

 Run both servers in parallel: Express on :3001, Laravel on :8000

 Verify all 22 public routes and 22 admin panel routes work with new backend
 Confirm session cookies, CSRF tokens, and Sanctum work correctly in production HTTPS

Deploy

 Remove server.ts, server/ directory, and Node.js backend dependencies from package.json

 Final check: APP_KEY set, all env secrets migrated (SMTP, Stripe, PayPal, session secret)

Step 1: Access Admin Dashboard
Navigate to /admin/login
Log in with admin credentials
Go to Landing Management (/admin/landing or /admin/cms)
Step 2: Customize Hero Section
Admin Dashboard → Landing Management → Hero Tab

Background: Upload hero background image or video
Headline: Edit main heading text (e.g., "WHERE")
Subheading: Customize description text
CTA Buttons:
Configure "Start Your Journey" button (text, link, style)
Configure "Explore Clubs" button (text, link, style)
Overlay Settings: Adjust opacity and gradient colors
Step 3: Configure Navbar

Logo: Upload custom logo or use text logo

Step 4: Manage Content Sections
Admin Dashboard → Landing Management → Sections Tab

Events Section
Enable/disable section
Edit section title and subtitle
Configure display settings (grid layout, card style)
Link to Events Management for content
Clubs Section
Enable/disable section
Edit section title and description
Configure featured clubs display
Set number of clubs to show
Link to Clubs Management
Membership Section
Enable/disable section
Edit membership tiers:
Free Tier: Name, features, CTA text
Premium Tier: Name, price, features, CTA text
VIP Tier: Name, price, features, CTA text
Configure pricing display format
Testimonials Section
Enable/disable section
Add/edit/remove testimonials
Each testimonial: Author name, role, content, photo
Configure carousel settings

Best Practices
Always backup before making major changes
Test changes in staging/preview mode first
Optimize images before uploading (recommended: WebP format)
Use consistent naming for better organization
Regular updates - keep content fresh
Monitor analytics - understand what works
Mobile-first - always check mobile view
Accessibility - use alt texts, proper heading hierarchy
Quick Setup Workflow
1. Set up Navbar & Logo
2. Customize Hero Section
3. Add Clubs (minimum 3)
4. Add Events/Activities (minimum 5)
5. Create Blog Posts (minimum 3)
6. Upload Gallery Images
7. Configure Contact Information
8. Set Up Payment Integration (if needed)
9. Test all links and forms
10. Launch!
